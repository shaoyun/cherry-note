## 快速开始

### 安装应用
- **Android**: 从应用商店下载安装APK文件
- **Windows**: 下载ZIP文件，解压后运行CherryNote.exe
- **macOS**: 下载DMG文件，拖拽到应用程序文件夹

### 首次设置
1. 启动应用后，点击设置按钮
2. 配置S3存储连接信息：
   - 端点URL (Endpoint)
   - 访问密钥 (Access Key)
   - 密钥 (Secret Key)
   - 存储桶名称 (Bucket Name)
3. 点击"测试连接"确保配置正确
4. 保存设置并开始使用

## 界面介绍

Cherry Note 采用经典的三栏布局设计：

```
┌─────────────┬─────────────┬─────────────────────┐
│             │             │                     │
│  文件夹树   │  笔记列表   │    编辑预览区       │
│             │             │                     │
│  - 工作笔记 │ □ 项目计划  │ # 标题              │
│    - 项目A  │ □ 会议记录  │                     │
│    - 项目B  │ □ 技术文档  │ 这里是笔记内容...   │
│  - 个人笔记 │             │                     │
│             │             │ [预览] [编辑]       │
│  标签过滤   │             │                     │
│  #工作 #学习│             │                     │
└─────────────┴─────────────┴─────────────────────┘
```

### 左侧栏 - 文件夹树和标签
- **文件夹树**: 显示所有笔记的层级结构
- **标签过滤**: 显示所有可用标签，点击可过滤笔记

### 中间栏 - 笔记列表
- 显示当前选中文件夹下的所有笔记
- 支持搜索和排序功能
- 显示笔记标题、修改时间和标签

### 右侧栏 - 编辑预览区
- **编辑模式**: Markdown编辑器，支持语法高亮
- **预览模式**: 实时渲染的Markdown预览
- **分屏模式**: 同时显示编辑和预览

## 基本功能

### 创建笔记
1. 右键点击文件夹树中的文件夹
2. 选择"新建笔记"
3. 输入笔记标题
4. 开始编写内容

### 编辑笔记
1. 在笔记列表中点击要编辑的笔记
2. 在右侧编辑区域输入Markdown内容
3. 内容会自动保存

### 组织笔记
- **创建文件夹**: 右键文件夹树 → 新建文件夹
- **移动笔记**: 拖拽笔记到目标文件夹
- **删除笔记**: 右键笔记 → 删除

### 添加标签
1. 在编辑器顶部的标签输入框中输入标签
2. 按回车键添加标签
3. 点击标签旁的×删除标签

## 高级功能

### Markdown语法支持
Cherry Note 支持完整的Markdown语法：

- **标题**: `# 一级标题` `## 二级标题`
- **强调**: `**粗体**` `*斜体*`
- **列表**: `- 无序列表` `1. 有序列表`
- **链接**: `[链接文本](URL)`
- **图片**: `![图片描述](图片URL)`
- **代码**: `` `行内代码` `` 和 ``` 代码块 ```
- **表格**: 使用 `|` 分隔列
- **引用**: `> 引用内容`

### 便签功能
1. 点击工具栏的"便签"按钮
2. 快速创建临时笔记
3. 便签会自动保存到"便签"文件夹

### 搜索功能
- **全文搜索**: 在搜索框中输入关键词
- **标签过滤**: 点击左侧的标签进行过滤
- **文件夹过滤**: 选择特定文件夹查看其下的笔记

### 导入导出
- **导出**: 文件 → 导出 → 选择格式和位置
- **导入**: 文件 → 导入 → 选择要导入的文件或文件夹

## 设置配置

### S3存储配置
在设置页面配置云存储连接：

- **端点URL**: S3兼容服务的端点地址
- **访问密钥**: 您的Access Key ID
- **密钥**: 您的Secret Access Key
- **存储桶**: 用于存储笔记的存储桶名称
- **区域**: 存储桶所在区域（可选）

### 应用设置
- **主题**: 选择浅色或深色主题
- **字体大小**: 调整编辑器字体大小
- **自动同步**: 开启/关闭自动同步功能
- **同步间隔**: 设置自动同步的时间间隔

### 编辑器设置
- **实时预览**: 开启/关闭实时预览
- **语法高亮**: 开启/关闭语法高亮
- **自动保存**: 设置自动保存间隔
- **分屏模式**: 选择默认的编辑器布局

## 常见问题

### Q: 如何同步我的笔记到其他设备？
A: 配置相同的S3存储设置，应用会自动同步所有笔记和文件夹结构。

### Q: 笔记数据存储在哪里？
A: 笔记主要存储在您配置的S3兼容存储中，本地会保留缓存以支持离线使用。

### Q: 支持哪些文件格式？
A: 主要支持Markdown格式(.md)，也可以导入纯文本文件(.txt)。

### Q: 如何备份我的笔记？
A: 使用导出功能可以将笔记导出为ZIP文件进行备份。

### Q: 应用支持离线使用吗？
A: 是的，应用支持离线模式，离线时的更改会在联网后自动同步。

### Q: 如何处理同步冲突？
A: 当检测到冲突时，应用会提示您选择保留本地版本、远程版本或创建副本。

### Q: 可以插入图片吗？
A: 可以，支持插入网络图片链接，本地图片会自动上传到S3存储。

### Q: 如何更改应用语言？
A: 应用会自动检测系统语言，目前支持中文和英文。

## 快捷键

### 通用快捷键
- `Ctrl/Cmd + N`: 新建笔记
- `Ctrl/Cmd + S`: 保存笔记
- `Ctrl/Cmd + F`: 搜索
- `Ctrl/Cmd + ,`: 打开设置
- `Ctrl/Cmd + Q`: 退出应用

### 编辑器快捷键
- `Ctrl/Cmd + B`: 粗体
- `Ctrl/Cmd + I`: 斜体
- `Ctrl/Cmd + K`: 插入链接
- `Ctrl/Cmd + Shift + C`: 插入代码块
- `Ctrl/Cmd + Shift + P`: 切换预览模式

### 导航快捷键
- `Ctrl/Cmd + 1`: 聚焦文件夹树
- `Ctrl/Cmd + 2`: 聚焦笔记列表
- `Ctrl/Cmd + 3`: 聚焦编辑器
- `F11`: 全屏模式
