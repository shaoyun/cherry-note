## 版本说明

### 版本号规则
Cherry Note 遵循语义化版本规则：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 发布周期
- **主版本**：每年 1-2 次重大更新
- **次版本**：每季度功能更新
- **修订版本**：根据需要发布 Bug 修复

### 支持政策
- **当前版本**：完整功能支持和 Bug 修复
- **前一个主版本**：重要 Bug 修复和安全更新
- **更早版本**：仅安全更新

## 已知问题

### 当前版本限制
- 不支持实时协作编辑
- 不支持插件扩展
- 图片仅支持网络链接和本地上传
- 数学公式渲染功能有限

### 平台特定问题
- **Android**: 某些设备上的文件权限可能需要手动授权
- **Windows**: 首次运行可能被防病毒软件误报
- **macOS**: 需要在安全设置中允许运行未签名应用（开发版本）

### 从测试版升级

2. 重新配置 S3 连接设置
